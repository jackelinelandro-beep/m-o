# Prototipo final · Mercedes & Oscar

Versión de aprobación previa a la conexión con Supabase.

## Publicación en GitHub Pages

Sube **el contenido de esta carpeta** a la raíz del repositorio, de modo que
`index.html`, `styles.css`, `app.js` y `config.js` queden al mismo nivel.

En GitHub abre **Settings → Pages** y selecciona:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

La web funciona como prototipo sin Supabase. Los formularios usan únicamente
el navegador local y las fotografías permanecen bloqueadas hasta conectar el
backend definitivo.
